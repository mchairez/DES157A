Mini Challenge 03

For this challenge add script tags at the bottom of this page, just before the closing body tag, and add a script that defines a function called greeting, which takes one parameter. The parameter will contain the name of a person. Then in the script create a variable that creates a custom welcome greeting for that person and returns the value of that variable.

Then add a call to the console.log method that runs that function, passing in a name of a person, so you get the custom greeting displaying in the console, as seen in the screenshot below.
